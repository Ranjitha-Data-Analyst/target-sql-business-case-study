# target-sql-business-case-study
SQL analysis of Target Brazil ecommerce dataset with insights and recommendations

## Project Overview
This project analyzes Target Brazil's e-commerce dataset (2016–2018) to extract business insights.

## Dataset
The dataset contains:
- customers
- sellers
- orders
- order_items
- payments
- reviews
- products
- geolocation

## Analysis Performed
- Order trends over time
- Monthly seasonality
- Customer distribution by state
- Freight and delivery analysis
- Payment method analysis

## Key Insights
- Most orders are placed in the afternoon
- São Paulo has the highest customer base
- Lower freight cost states perform better
- Credit card installment payments dominate

## Tools Used
- SQL
- BigQuery
- Data Analysis
